# Portfolio Generator (Streamlit)

Generátor statických portfolií s více šablonami a GitHub integrací.

## Funkce

- **4 šablony**: Minimal, Gradient, Cards, Terminal
- **GitHub import**: Automatické stažení repozitářů
- **Live preview**: Okamžitý náhled změn
- **Export HTML**: Stažení hotového portfolia

## Spuštění lokálně

```bash
cd portfolio-streamlit
pip install -r requirements.txt
streamlit run app.py
```

## Deployment na Streamlit Cloud

1. Push to GitHub
2. Go to [streamlit.io/deploy](https://streamlit.io/deploy)
3. Connect your GitHub repo
4. Deploy!

## Šablony

| Šablona | Popis |
|---------|-------|
| Minimal | Tmavý, čistý design |
| Gradient | Gradient pozadí, glassmorphism |
| Cards | Světlý, card-based layout |
| Terminal | Retro terminal styl |
